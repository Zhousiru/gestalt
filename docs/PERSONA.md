# Writing a Persona

Put persona files in `GestaltHome/persona/`.

Recommended layout:

```text
persona/
  1-identity.md
  2-style.md
  3-interests.md
  4-cognitive.md
  5-boundaries.md
```

This layout is optional; a single file also works. Gestalt loads `.md`, `.markdown`, `.txt`, and `.jsonl` files in filename order. Restart Gestalt after editing them.
